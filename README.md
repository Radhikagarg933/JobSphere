# 💼 JobSphere – Full-Stack Recruitment Platform

## 📌 Overview

JobSphere is a full-stack recruitment platform built using Django and MySQL that streamlines the hiring process for both job seekers and employers. The platform enables employers to post and manage job opportunities while allowing candidates to search, apply, and track applications through a secure and user-friendly interface.

## 🚀 Features

### 👤 Authentication & User Management

* Role-Based Authentication (Job Seeker & Employer)
* Secure User Registration and Login
* Email OTP Verification
* Profile Management

### 💼 Job Management

* Create, Update, and Manage Job Listings
* Browse Available Jobs
* Apply for Jobs
* Application Status Tracking

### 📊 Dashboards

* Dedicated Employer Dashboard
* Dedicated Job Seeker Dashboard
* Manage Posted Jobs and Applications
* View Application History

### 📧 Notifications

* OTP Verification Emails
* Automated Application Status Notifications
* Real-Time Email Updates using Django Signals

## 🔐 Security Features

* Password Hashing
* CSRF Protection
* SQL Injection Prevention via Django ORM
* Secure Session Management
* Email-Based OTP Verification
* Form Validation and Input Sanitization

## 🛠️ Tech Stack

### Frontend

* HTML5

### Backend

* Python

* MySQL

### Tools & Libraries

* Django ORM
* Django Signals
* SMTP Email Service

## ⚙️ How It Works

1. Users register and verify their accounts through Email OTP.
2. Employers post and manage job opportunities.
3. Job seekers browse and apply for jobs.
4. Employers review applications and update statuses.
5. Candidates receive automated email notifications regarding application updates.

## ▶️ Run the Project

```bash
python manage.py runserver
```

Open your browser and visit:

```text
http://127.0.0.1:8000/
```
